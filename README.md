# CODSOFT
This repository contains the tasks and projects completed as part of my CodSoft Python Programming Internship. Each task demonstrates my learning and practice in Python programming.
## Task 1 – QuickCalc (Simple Calculator)

QuickCalc is a simple Python calculator that performs basic arithmetic operations.  
It allows users to enter two numbers and choose an operation to get the result.

### Features
- Addition
- Subtraction
- Multiplication
- Division
- Handles division by zero
- Allows users to perform multiple calculations

### Technologies Used
- Python 3
- Visual Studio Code (VS Code)
- Command Line Interface (CLI)
